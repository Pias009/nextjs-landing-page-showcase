## Admin Panel Features

Once logged in, you can:

### 📊 Dashboard
- View statistics (total pages, views, categories)
- See all landing pages in a table format

### ➕ Add New Landing Page
- Click "Add New Page" button
- Fill in the form:
  - **Title**: Name of your landing page
  - **Description**: Brief description
  - **Image URL**: Link to preview image
  - **Live URL**: Link to the live landing page
  - **Category**: Select from 7 categories
- Click "Add Page" to save

### ✏️ Edit Landing Page
- Click the **Edit** icon (pencil) on any page
- Update the information
- Click "Update Page" to save changes

### 🗑️ Delete Landing Page
- Click the **Delete** icon (trash) on any page
- Confirm the deletion
- Page will be removed immediately

### 👁️ View Live
- Click the **View Live** icon (eye) to open the landing page in a new tab

### ⚙️ Settings
- Click the **Settings** icon (gear) in the top right
- Update your admin username and password
- Click "Save Settings" to apply changes
- **Important**: Remember your new credentials!

### 🚪 Logout
- Click the **Logout** button to exit the admin panel

## Security Notes

- Credentials are stored in your browser's localStorage
- For production use, consider implementing backend authentication
- Change the default credentials after first login
- The admin panel is protected - users cannot access it without login
- Landing page source codes are NOT exposed to public users
